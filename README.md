# Таймер для стрельбы — диагностический рекордер

Веб-приложение для iPhone (Safari): записывает звук выстрелов, считает их по звуку,
строит график силы звука (dBFS) и спектрограмму. Детектор выстрелов с резкой эха.

## Запуск

Открыть в Safari по HTTPS-ссылке (GitHub Pages). Микрофону нужен HTTPS — с локального
файла не заработает.

1. «🎤 Включить микрофон» — выдать разрешение.
2. Выбрать тип (одиночный/двойной/серия), нажать «● Запись», стрелять, «■ Стоп».
3. Запись сохраняется в браузер (IndexedDB): сырой WAV + график + спектрограмма.
4. Кнопки у записи: графики, прослушать, скачать WAV/JSON, удалить.

## Детектор (4 слоя)

- L1 — фронт по крутизне (dB/мс), не по амплитуде → работает при клиппинге микрофона.
- L2 — refractory + окно эха по известной задержке.
- L3 — прогноз эха от каждого выстрела; резкий фронт в окне эха = реальный выстрел поверх эха.
- L4 — спектральный центроид: глухой звук = эхо, яркий = выстрел.

Зелёная метка — засчитан, красная — зарезано как эхо. Все пороги на ползунках.
Профили настроек сохраняются (несколько вариантов).

## Важно

iPhone-микрофон клиппит на выстреле (потолок ~120–125 dB SPL, выстрел 140–170).
Реальные dB SPL измерить нельзя — график относительный (dBFS). Детекция строится
на форме фронта и времени, не на громкости.

Дефолтные пороги — теоретические. Тюнятся под реальные записи после полевого теста.
